# Installation VoiceDesk

## Fichiers disponibles

### macOS
- **VoiceDesk_0.1.0_aarch64.dmg** - Installation pour Mac (Apple Silicon)
- Double-cliquez pour installer
- Faites glisser VoiceDesk vers Applications

### Windows
- Les fichiers Windows seront générés automatiquement via GitHub Actions
- Visitez: https://github.com/kais75sellami-ops/VoiceDesk/actions
- Téléchargez les artifacts après le build

## Comment générer les versions Windows

1. Allez sur GitHub: https://github.com/kais75sellami-ops/VoiceDesk
2. Cliquez sur l'onglet "Actions"
3. Sélectionnez "Build VoiceDesk Release"
4. Cliquez sur "Run workflow"
5. Une fois terminé, téléchargez les artifacts (VoiceDesk-Windows)
