================================================================================
                              QUICK START
================================================================================

1. INSTALL DEPENDENCIES:
   cd VoiceDesk
   npm install

2. RUN IN DEVELOPMENT:
   npm run tauri dev
   
   (Enter ElevenLabs API key when prompted)

3. BUILD FOR PRODUCTION:
   npm run tauri build
   
   macOS output: src-tauri/target/release/bundle/macos/VoiceDesk.app
   Windows output: src-tauri/target/release/VoiceDesk.exe

================================================================================
                              CUSTOMIZATION
================================================================================

CHANGE APP NAME:
  • Edit src-tauri/tauri.conf.json (productName)
  • Edit src-tauri/Cargo.toml (name)
  • Edit package.json (name)
  • Update window title in tauri.conf.json

CHANGE ICONS:
  1. Replace PNG files in icons/ directory
  2. Run: python3 generate_icons.py
  3. Rebuild app

ADD LANGUAGES:
  Edit src/types.ts and add to LANGUAGES array
